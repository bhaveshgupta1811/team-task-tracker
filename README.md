# \# Team Task Tracker

# 

# A full-stack web application to manage teams, tasks, and activity timelines.

# 

# \## Tech Stack

# \- Frontend: Next.js

# \- Backend: Django

# \- Database: MongoDB

# 

# \## Features

# \- User Authentication

# \- Team Management

# \- Task Assignment

# \- Activity Timeline

# \- Role-Based Access

# 

# \## Project Structure

# /backend → Django backend  

# /frontend → Next.js frontend
